# My Portfolio Website

This is my beginner portfolio website built using **HTML** and **CSS** as part of my frontend learning journey.

The project focuses on practicing basic web structure, navigation, forms, and styling while building a simple personal portfolio layout.

---

## Project Overview

This portfolio includes:

- A navigation bar with anchor links
- A home section introducing me
- An about section describing my interests
- A work section listing practice projects
- A resume section with a basic skills table
- A contact form
- A styled footer

This project helped me practice structuring a webpage using semantic HTML and improving visual appearance with CSS.

---

## Technologies Used

- HTML5  
- CSS3

---

## Features

- Inline navigation menu  
- Multiple page sections using anchor links  
- Styled form inputs with focus effects  
- Basic table styling  
- Clean spacing and layout using CSS
